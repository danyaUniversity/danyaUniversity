Падав сніг на поріг
Платон Воронько

Падав сніг на поріг.
Кіт зліпив собі пиріг.
Поки смажив, поки пік,
а пиріг — водою стік.
Кіт не знав, що на пиріг
треба тісто, а не сніг.
